## Karteikarten System {.smaller}

::: {style="display: flex; gap: 2rem; align-items: center;"}

::: {style="flex: 1;"}
### Fächer System
- **Fach 1**: Täglich abfragen
- **Fach 2**: Jeden zweiten Tag  
- **Fach 3**: Einmal die Woche
- **Fach 4**: Alle 30 Tage
:::

::: {style="flex: 1;"}
### Ablauf
```
neu → 1 → 2 → 3 → 4
     ↓   ↓   ↓   ↓
   falsch richtig richtig richtig
```
:::

:::

---

## Funktionsweise {.smaller}

::: {style="display: flex; gap: 2rem;"}

::: {style="flex: 1;"}
### Karten Format
- **Vorderseite**: Die Frage
- **Rückseite**: Die Antwort

### Bewegung der Karten
- **Richtig beantwortet** → nächstes Fach
- **Falsch beantwortet** → zurück zu Fach 1
:::

::: {style="flex: 1;"}
### Vorteile
- Spaced Repetition Prinzip
- Effizientes Lernen
- Fokus auf schwierige Karten
- Systematische Wiederholung

### Anwendung
Ideal für:
- Programmierkonzepte
- Datenbankbegriffe  
- Syntax und Befehle
:::

:::

---

## Lessons Learned

::: {.incremental}
- Regelmäßige Wiederholung ist der Schlüssel zum Erfolg
- Schwierige Karten werden häufiger wiederholt
- Das System passt sich automatisch an den Lernfortschritt an
- Langfristiges Behalten durch optimierte Intervalle
:::

---

## Zusammenfassung

Das Karteikarten-System mit **4 Fächern** ermöglicht:

- Systematisches Lernen durch **Spaced Repetition**
- Automatische Anpassung an individuelle Schwächen
- Optimale Nutzung der Lernzeit
- Nachhaltiges Einprägen von Wissen

:::{.callout-tip}
## Tipp
Beginnen Sie mit wenigen Karten und erweitern Sie das System schrittweise!
:::
